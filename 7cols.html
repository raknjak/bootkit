SEVEN COLUMNS

As there are seven days in one week we need a custom column structure that will come in handy when creating date-related structured like for example weather widgets.

Available classes

  • bk-seven-cols: adds 7 column behavior to a row
  • col-sm-1: is used to make the columns float left except for small screens
  • col-xs-1: columns always float when using this class
