CAROUSEL

Bootstrap carousel can be used to display moving or fading widgets.

Available classes

  • bk-carousel-ctrl: custom design for the carousel controls to remove default Bootstrap carousel styles.
  • bk-carousel-hide-init: add this class to controls which need to be hidden unless when hovering over the carousel.
  • bk-custom-ctrl: add this class to controls which need to be hidden unless when hovering over the carousel.
  • bk-fade: replace the slide class with this one to have slides (items) fade instead.
