OPACITY

A very handy way to make colors appear lighter or darker or to have opacity hover effects.

Available classes

  • bk-opacity: sets the default opacity to .8 (can be redefined in LESS) and on hover the opcaity becomes 1.
  • bk-opacity-inv: sets the default opacity to 1 and on hover the opcaity becomes .8 (can be redefined in LESS).
  • bk-opacity-2: sets the opacity of the target element to .2
  • bk-opacity-4: sets the opacity of the target element to .4
  • bk-opacity-5: sets the opacity of the target element to .5
  • bk-opacity-6: sets the opacity of the target element to .6
  • bk-opacity-8: sets the opacity of the target element to .8
  • bk-opacity-light: an overlay which sets background: rgba(255,255,255,0.1) to a wrapper element
  • bk-opacity-dark: an overlay which sets background: rgba(0,0,0,0.1) to a wrapper element

We need to fix some Chrome on Window annoyances: use bk-chrome-fix whenever the hover effect breaks rounded corners, image moving bugs appear or unwanted element movements are detected.
